This extension checks playlists for videos that are missing/unavailable, and makes the process of replacing those videos much easier.

Because of how the YT API works, the account needs to have a channel associated with it. You don't need to create a video or anything extra, the channel just needs to exist for the extension to work properly.

Also, YouTube music playlists don't appear to be distinguishable from regular playlists, so if you have any of those, the error counts for those playlists may be inaccurate due to music added locally to a playlist (in other words, the extension can't find the song if it's not one from YouTube as a platform).

Button Info:
"Extension Info" - displays this page
"Check Playlists" - displays all playlists you own, the number of videos in each playlist, and the number of videos unable to be found.
